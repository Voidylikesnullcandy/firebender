# Firebender - GTA V Enhanced

Hi, this is my first GTA V Enhanced mod.
I would really love feedback, bugs encountered and contributions from the community.
I would also like to learn from you guys as well.
So feel free to interact. :)

## Requirements
- GTA V Enhanced (legal copy)
- ScriptHookV by Alexander Blade

## Install
Drop `Firebender.asi` into your GTA V root directory.

## Usage

> ⚠️ **Story Mode only.** Do not use in GTA Online.

### Keyboard

| Key | Action |
|-----|--------|
| `J` | Enable / Disable Firebending |
| `T` | Flame Force |
| `Y` | Throw Fireball |
| `U` | Fire Spark |
| `I` | Fire Grenade |

### Controller

| Combo | Action |
|-------|--------|
| `L1 + R2` *(quick tap)* | Enable / Disable Firebending |
| `L1 + X` | Flame Force |
| `L1 + Square` | Throw Fireball |
| `L1 + Triangle` | Fire Spark |
| `L1 + Circle` | Fire Grenade |

> 💡 For the enable/disable combo, tap quickly as holding `L1` too long will open the weapon wheel.

## Legal
This mod uses only ScriptHookV's public scripting API.
No Rockstar game files are included or were reverse engineered.
